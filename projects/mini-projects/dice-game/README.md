# 🎲 Python Dice Game

A fun and beginner-friendly **Dice Game** built using **Python**.

This project simulates rolling dice using random number generation and is a great beginner project to learn Python basics like loops, conditions, functions, and modules.

---

## ✨ Features

* 🎲 Roll Virtual Dice
* 👥 Single Player / Multiplayer Support
* 🔢 Random Number Generation (1 to 6)
* 🏆 Winner Decision Logic
* 🔁 Play Again Option
* 🖥️ Console Based Gameplay

---

## 🛠️ Technologies Used

* Python 3
* random Module

---

## 🎮 Example Gameplay

```text id="85hzif"
Player 1 rolled: 5
Player 2 rolled: 3

Player 1 Wins!
```

## 🎯 Concepts Practiced

* Loops
